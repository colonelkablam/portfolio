KablamGame 3D

Retro Style Raycaster Shooter - using my console graphics engine

The game renders the 3D environment using Digital Differential Analysis (DDA) raycasting. This is possible as the world is represented as a tile-based map (essentially a 2D array of integers). There are textures for walls, floors, ceilings, and sprites, with a depth buffer system to handle rendering order and visibility, a dynamic skybox, and sound effects managed by a custom sound manager (implementing the irrKlang library).

Key features include:

  • Graphics: The raycasting system iterates through each column of the screen, calculating the angle and distance to walls, floors, and ceilings. The system then uses this to determine the intersection points with the environment and sample the textures accordingly. Different display settings (e.g., normal, distance shading, angle shading) are supported, allowing for various visual effects. I am currently working on a more advanced lighting solution and the challenge is making it run at a playable speed.
  • Texture Management:. Efficient handling and rendering of textures onto a pseudo 3D environment.
  • Sprite Management: Using the composite design pattern, sprites are composed of multiple smaller components. This makes it easier to manage and extend their behavior for the desired effect.
  • Collision Detection: line intersection collision detection algorithms are used for interactions between the player, environment, and other objects.
  • AI and Object Interaction: Implemented AI behaviors, including enemy characteristics and their interaction with the player and the environment.
  • Sound Management: Integrated sound manager for in-game music and sound effects, enhancing the immersive experience using the irrKlang library rather than Windows for more comprehensive sound management.

    • This project has been a wonderful way to increase my ability to develop complex systems from the ground up, and taught me a lot about computer graphics, game mechanics, and software engineering principles in the process. It continues to be great fun to develop.

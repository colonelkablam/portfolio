TexturePainter Application

GUI texture editor app built using the console graphics engine

This project arose due to the need to draw textures to be used within my graphics engine. It ended up taking a lot longer and being more complex than first intended. It became apparent that if I wanted to design some interesting sprites, hard coding 2D arrays with numbers was going to be very frustrating and take a long time. Ultimately this editor has a pretty high level of functionality for drawing/copying/editing the textures to be used by the engine and allows for copying and pasting between sprites, filling, flipping, cutting and shape drawing.

Helpful Resources and Content

OneLoneCoder - An extreamly useful resource for learning the Windows API and handling the console. After some flondering with the screen buffer it was very fortunate to find his tutorial series! His texture format was used in my design. I also based my game-loop implementation on his initial design.
3DSage - A very good set of tutorials for implementing raycasting. This was particularly helpful for rendering the floor and ceiling textures.
Lode's DDA raycasting tutorial - A particularly thorough explanation of raycasting.

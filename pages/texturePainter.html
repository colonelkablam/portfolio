TexturePainter Application

GUI texture editor app built using the console graphics engine

This project arose due to the need to draw textures to be used within my graphics engine. It ended up taking longer and becoming more complex than first intended. It became apparent that if I wanted to design some interesting sprites, hard coding 2D arrays with numbers was going to be very frustrating and take a long time!

Ultimately this editor ended up with a pretty high level of functionality for drawing/copying/editing the textures to be used by the engine. It allows for copying and pasting between sprites, filling, flipping, cutting, line and shape drawing. The user can also zoom in and pan around the canvas to make more detailed drawing easier.

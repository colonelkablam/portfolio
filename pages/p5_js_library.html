Various p5.js Library Projects

Line Line Intersection Raycasting

A small interactive p5.js sketch showing particles emitting rays and using line-line intersection to calculate their collision with walls.


Link to the sketch on the p5.js site

M.O.T.H

A small single player game about saving moths from city street lights and letting them escape to the moon (some artistic licence has been taken regarding the true behaviour of moths).


Link to the M.O.T.H game on the p5.js site

Santa vs Rudolf

Santa and Rudolf compete head-to-head to collect as many presents as posible after a sledge crash in a forest infested with zombie snowpeople.


Link to the Santa vs Rudolf on the p5.js site

Helpful Resources and Content

TheCodingTrain - A wonderful educational resource for creative coding using Processing and p5.js library. Particularly good for artistic inspiration and guidence for simulations.

Processing.org - An amazing set of resources. Basically a flexible software sketchbook and a language for learning how to code

The Nature of Code - The Nature of Code is an amazing book by Daniel Shiffman. In his words this resource is for anyone who "Ever wondered how the unpredictable evolutionary and emergent properties of nature can be captured in a digital canvas, all through the magic of code?"
